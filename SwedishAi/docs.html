Kom igång

SwedishAI erbjuder ett kraftfullt API för naturlig språkbehandling på svenska. Vårt API ger dig tillgång till state-of-the-art språkmodeller optimerade för svenska.

Autentisering

curl https://api.swedishai.com/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Sammanfatta denna text..."}'

API Referens

Text Generation

Generera text baserat på en prompt.

Parametrar

  • prompt (string): Text att utgå ifrån
  • max_tokens (int): Max antal tokens att generera (default: 100)
  • temperature (float): Kreativitet i genereringen (0.0-1.0)
  • top_p (float): Nucleus sampling parameter (0.0-1.0)

Exempel respons

{
    "text": "Genererad text här...",
    "usage": {
        "prompt_tokens": 10,
        "completion_tokens": 50,
        "total_tokens": 60
    }
}

Exempel

Textgenerering


import requests

response = requests.post(
    "https://api.swedishai.com/v1/generate",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json={
        "prompt": "Skriv en sammanfattning om AI i Sverige",
        "max_tokens": 100
    }
)

print(response.json()["text"])
